# DEFINITIVE_BOX2D_SDL_DEV

**How to play**

A = Left Movement
D = Right Movement
SPACE = Jump
Up Arrow = Change gravity to upwards
Down Arrow = Change gravity to downwards
F8 = Change from 60 to 30 fps
F1 = God mode/Debug
ESC = Close game

Win condition = Arrive to the Strawberry
Lose condition = Die
